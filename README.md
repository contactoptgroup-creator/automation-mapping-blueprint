# Automation Mapping Blueprint

Guide interactif pour mapper et identifier les automatisations a fort ROI.

## Description

Presentation HTML interactive basee sur le cours "10 Hours to 10 Seconds V2" de la AI Automation Society+. Utilisable comme cheat sheet en clientele pour guider le processus de mapping d'automatisations.

## Fonctionnalites

- **3 Phases completes:** Personal Leverage, Mapping, Client Work
- **Barre de progression:** Track de l'avancement par projet
- **Multi-projets:** Gestion de plusieurs clients/projets avec sauvegarde locale
- **Checklists interactives:** Validation etape par etape
- **Design moderne:** Dark mode, animations, responsive

## Utilisation

1. Ouvrir `Automation-Mapping-Blueprint.html` dans un navigateur
2. Creer un nouveau projet (nom du client/projet)
3. Suivre les etapes phase par phase
4. Cocher les checklists au fur et a mesure
5. La progression est sauvegardee automatiquement

## Framework

Base sur le framework "10 Hours to 10 Seconds":
- **Time Lens:** Identifier ou le temps est perdu
- **Income Lens:** Calculer le ROI potentiel
- **ICE Scoring:** Prioriser (Impact, Confidence, Ease)
- **Wireframing:** Dessiner le workflow avant de construire
- **Context Engineering:** Donner le bon contexte a l'IA
